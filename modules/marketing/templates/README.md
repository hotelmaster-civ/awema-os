# Templates réutilisables — Marketing

Gabarits vierges à **dupliquer** pour démarrer vite et rester cohérent d'un client à
l'autre. Ne pas remplir ces fichiers : les **copier** dans le dossier client.

| Fichier | Usage |
|---|---|
| `template-persona.md` | Fiche persona |
| `template-fiche-contenu.md` | Une fiche de contenu social complète |
| `template-script-video.md` | Script Reel/TikTok/Short |
| `template-calendrier.csv` | En-têtes du calendrier éditorial |
| `template-scoring.csv` | En-têtes + formules du scoring |

> Les colonnes et formules sont documentées dans chaque template.
